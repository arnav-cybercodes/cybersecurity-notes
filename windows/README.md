# Windows Fundamentals - Extended Notes

## Why Windows is Important in Cybersecurity

Windows is the most widely used desktop operating system in enterprises and homes. Because of its popularity, it is one of the primary targets for:

- Malware
- Ransomware
- Phishing attacks
- Privilege Escalation attacks
- Credential Theft
- Remote Access Trojans (RATs)

Understanding Windows internals helps security professionals investigate incidents and secure systems.

---

# Windows Architecture

Windows consists of two major modes:

## User Mode

Applications run here.

Examples:
- Google Chrome
- Microsoft Word
- Notepad
- Calculator

If an application crashes in User Mode, the operating system usually continues running.

## Kernel Mode

Core operating system components run here.

Examples:
- Device Drivers
- Memory Manager
- Process Scheduler

Errors in Kernel Mode can result in:

- Blue Screen of Death (BSOD)
- System crashes
- Hardware failures

# Environment Variables

Environment variables store system information and paths.

Useful Variables:

```text
%SystemRoot%
%WinDir%
%UserProfile%
%Temp%
%AppData%
```

Example:

```text
%UserProfile%\Desktop
```

Points to the current user's desktop.

---

# Windows Processes

A process is a running instance of a program.

Examples:

- explorer.exe
- chrome.exe
- notepad.exe
- powershell.exe

View processes using:

```powershell
Get-Process
```

or

```text
Task Manager
```

---

# Services

Services run in the background and often start automatically.

Examples:

- Windows Update
- Print Spooler
- DHCP Client
- Windows Defender

View Services:

```powershell
Get-Service
```

---

# Registry

The Windows Registry is a database that stores configuration settings.

Major Registry Hives:

| Hive | Purpose |
|--------|---------|
| HKLM | System-wide settings |
| HKCU | Current user settings |
| HKCR | File associations |
| HKU | User profiles |
| HKCC | Current hardware profile |

Open Registry Editor:

```text
regedit
```

⚠️ Incorrect registry modifications may damage the operating system.

# Windows Security Features

## Windows Defender

Built-in antivirus solution.

Functions:
- Real-time protection
- Malware detection
- Virus scanning
- Threat removal

---

## Windows Firewall

Controls incoming and outgoing network traffic.

Benefits:

- Blocks unauthorized access
- Filters network connections
- Protects devices from attackers

Open Firewall Settings:

```text
wf.msc
```

---

## BitLocker

Full disk encryption feature.

Benefits:

- Protects sensitive data
- Prevents unauthorized access
- Useful if a device is stolen

---

# Event Viewer

Event Viewer stores logs generated by Windows.

Open:

```text
eventvwr.msc
```

Important Log Categories:

- Application
- Security
- Setup
- System

Security analysts frequently use Event Viewer during investigations.

# PowerShell

PowerShell is Microsoft's command-line shell and scripting language.

Useful Commands:

### Current User

```powershell

```

### List Processes

```powershell
Get-Process
```

### List Services

```powershell
Get-Service
```

### Network Connections

```powershell
Get-NetTCPConnection
```

### IP Configuration

```powershell
ipconfig
```
